# [Object Types](https://www.typescriptlang.org/docs/handbook/2/objects.html)

## Property Modifiers

针对属性可以设置：
- 类型
- 是否可选
- 是否只读

```ts
interface SomeType {
  type: string
  optional?: boolean
  readonly readonly: boolean
}
```


## Index Signatures

索引签名：应用于不知道对象所有属性名的场景

```ts
interface StringArray {
  [index: number]: string;
}
```
